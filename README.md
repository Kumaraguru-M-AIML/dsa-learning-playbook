The ecosystem is distributed into highly atomic, specialized functional clusters:

## 🚀 Running the System

Because of the new highly-modular architecture, a universal entry point has been placed at the root level to manage your `PYTHONPATH` automatically.

**Standard Execution Pattern:**
```bash
python run.py [module_name] [optional_args]
```

**Examples:**
*   `python run.py cq_mythos_v2` (Launches the engine)
*   `python run.py ablation_engine` (Triggers verification matrix)
