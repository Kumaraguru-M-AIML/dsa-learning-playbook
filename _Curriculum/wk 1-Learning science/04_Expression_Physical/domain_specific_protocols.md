# Domain-Specific Learning Protocols

> **Status**: Deepened R&D Reference Document
> **Purpose**: Tailored application of the Master Protocol for specific types of learning challenges.
> **Goal**: To translate abstract learning science into concrete field manuals.

---

## 1. THE CODING PROTOCOL (Procedural + Logic)

**Key Challenge**: Transforming abstract logic into rigid syntax, debugging, and system design.

### Phase A: Syntax Acquisition (The Basics)
*   **Technique**: **Spaced Repetition (Anki)** for syntax rules, elementary functions, and standard libraries.
*   **Technique**: **Micro-Drilling**. Write the same `for` loop or `class` structure 5-10 times until muscle memory sets in.

### Phase B: Logic & Problem Solving
*   **Technique**: **Parsons Problems**. Arrange mixed-up lines of code into correct order (focus on logic, not syntax).
*   **Technique**: **Rubber Ducking**. Explain line-by-line what code does to an inanimate object.
*   **Technique**: **Tracing**. Manually "run" code on paper, tracking variable states.

### Phase C: System Design
*   **Technique**: **Scaffolding**. Start with "pseudo-code" (comments) first, then fill in logic.
*   **Technique**: **Reverse Engineering**. Read open-source code. Analyze *why* they structured it that way.

## 2. THE LANGUAGE PROTOCOL (Declarative + Procedural)

**Key Challenge**: Massive vocabulary retention + rapid real-time processing (speaking/listening).

### Phase A: Vocabulary & Grammar
*   **Technique**: **Spaced Repetition (SRS)**. Critical frequency: Words 1-1000 (85% of communication).
*   **Technique**: **Dual Coding**. Image + Audio + Text on every flashcard.

### Phase B: Comprehension (Input)
*   **Technique**: **Comprehensible Input (i+1)**. Listen/Read material slightly above current level (80-90% understood).
*   **Technique**: **Shadowing**. Listen to audio, repeat immediately (0.5s delay) to mimick prosody.

### Phase C: Production (Output)
*   **Technique**: **Circumlocution**. Don't look up unknown words; describe them using words you *do* know.

## 3. THE CONCEPTUAL SCIENCE PROTOCOL (Abstract Systems)

**Key Challenge**: Understanding invisible forces and counter-intuitive subjective models (Physics, Bio, Math).

### Phase A: Model Construction
*   **Technique**: **Analogical Reasoning**. "Current is like water." (Voltage=Pressure, Amps=Flow).
*   **Technique**: **Visuospatial Sketching**. Draw the system. If you can't draw it, you don't understand it.

### Phase B: Testing & Refinement
*   **Technique**: **Elaborative Interrogation**. Ask "Why?" 5 times.
*   **Technique**: **Feynman Technique**. Teach the mechanism without jargon.

## 4. THE HISTORY/FACT PROTOCOL (Narrative + Data)

**Key Challenge**: Memorizing vast amounts of dates, names, and causal chains.

*   **Technique**: **Memory Palace**. Place events along a physical route in your mind.
*   **Technique**: **Narrative Chaining**. Turn dry facts into a causal story.
*   **Technique**: **Comparison Matrices**. Create table comparing "French Revolution" vs "American Revolution" (Cause, Key Figures, Outcome).

---

## 5. THE CREATIVE ARTS PROTOCOL (Subjective Production)

**Key Challenge**: Improvement in fields where "Right/Wrong" is subjective (Painting, Music, Writing).

### Phase A: Technical Constraint
*   **Technique**: **Constraint-Based Practice**. Do NOT "just paint." Paint using *only* 3 colors. Write a story *without* adjectives. Limits force creativity.
*   **Technique**: **Master Copying**. Replicate a masterwork stroke-for-stroke. This downloads the master's decision-making process into your muscles.

### Phase B: The Feedback Loop
*   **Technique**: **The Critique Group**. You need external eyes to spot "blind spots."
*   **Technique**: **Iterative Drafting**. Never accept the first draft. The "Art" is in the revision, not the creation.

---

## 6. THE SOFT SKILLS PROTOCOL (Social Dynamics)

**Key Challenge**: High variability. No two social interactions are identical.

### Phase A: Scripting
*   **Technique**: **Mental Rehearsal**. Script out the negotiation/leadership speech. Visualize the reaction.
*   **Technique**: **Pattern Recognition**. Study interactions (movies, meetings) and label the dynamics (e.g., "That was a Status Play," "That was Active Listening").

### Phase B: Low-Stakes Simulation
*   **Technique**: **Roleplay**. Practice the difficult conversation with a peer.
*   **Technique**: **Video Analysis**. Record yourself speaking. Watch it with sound OFF (body language). Watch it with screen OFF (tonality).

---

## 7. THE KINETIC PROTOCOL (Sports/eSports)

**Key Challenge**: Millisecond timing and proprioception.

### Phase A: Mechanics (Slow Motion)
*   **Technique**: **Super-Slow Practice**. Perform the movement at 10% speed. If you stumble at slow speed, you have a technique flaw that speed is hiding.
*   **Technique**: **Proprioceptive Tuning**. Close your eyes. Perform the move. Feel the limb position.

### Phase B: Game Speed
*   **Technique**: **Video Review**. Compare your footage side-by-side with a pro. The gap is often in *preparation* for the move, not the move itself.
