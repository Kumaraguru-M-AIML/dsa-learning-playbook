## 1. THE DAILY ARCHITECTURE (The 24h Cycle)

This schedule leverages the brain's natural circadian rhythms to optimize Encoding (Morning), Analysis (Mid-day), and Consolidation (Sleep).

### 🌅 Phase 1: Priming & High-Load Encoding (07:00 - 11:00)
*Target State: High Norepinephrine / High Dopamine (Alert & Focused)*

*   **07:00 | Protocol: The Wake-Up Cascade**
    *   **Action**: Sunlight (10min) + Hydration.
    *   **Neuroscience**: Sets the SCN timer for tonight's sleep (Melatonin onset).
    *   **Ref**: [`accelerated_learning_science.md`](file:///E:/wk%201-Learning%20science/accelerated_learning_science.md)
*   **08:00 | Deep Work Block 1 (The "Eat the Frog" Block)**
    *   **Focus**: **Acquisition of New Material** (Input).
    *   **Technique**: **Feynman Technique** + **Dual Coding**.
    *   **Why**: Prefrontal Cortex is freshest. Anterior Cingulate is ready for conflict detection (learning).
    *   **Ref**: [`skills_memory_encoding_deep_dive.md`](file:///E:/wk%201-Learning%20science/skills_memory_encoding_deep_dive.md)
*   **10:00 | Break Protocol: NSDR / Walking**
    *   **Action**: 15min Non-Sleep Deep Rest or Optic Flow walk.
    *   **Why**: Flushes adenosine, allows "Spontaneous Replay" of just-learned material.

### ☀️ Phase 2: Analysis & Problem Solving (12:00 - 16:00)
*Target State: Balanced Serotonin / Acetylcholine (Calm Focus)*

*   **13:00 | Deep Work Block 2 (The "Output" Block)**
    *   **Focus**: **Problem Solving & Skill Construction**.
    *   **Technique**: **Deliberate Practice** (Zone of Proximal Development).
    *   **Protocol**:
        1.  Attempt problem.
        2.  Fail.
        3.  **Diagnostic Pause** (Metacognition: "Why did I fail?").
        4.  Retry.
    *   **Ref**: [`skills_skill_construction_deep_dive.md`](file:///E:/wk%201-Learning%20science/skills_skill_construction_deep_dive.md)
*   **15:00 | The "Nap" Option (Consolidation)**
    *   **Action**: 20min Power Nap.
    *   **Why**: Clears hippocampus buffer to allow more encoding in the evening.

### 🌙 Phase 3: Retrieval & Consolidation (18:00 - 22:00)
*Target State: Low Cortisol / High Melatonin Preparation*

*   **19:00 | Active Recall Session**
    *   **Technique**: **Anki Spaced Repetition**.
    *   **Focus**: Reviewing material from *previous* days (Interleaving).
    *   **Why**: Strengthening synaptic weights via Retrieval-Induced Forgetting dynamics.
    *   **Ref**: [`skills_memory_retrieval_deep_dive.md`](file:///E:/wk%201-Learning%20science/skills_memory_retrieval_deep_dive.md)
*   **21:30 | The Sleep Gate**
    *   **Action**: No screens. Cool room.
    *   **Neuroscience**: Critical for **SWS-REM Cycling**. Phase 2 Spindles will move today's data from Hippocampus to Cortex.
    *   **Ref**: [`skills_retention_control_deep_dive.md`](file:///E:/wk%201-Learning%20science/skills_retention_control_deep_dive.md)

## 2. THE LEARNING ALGORITHM (Decision Tree)

Use this logic to decide which tool to use.

#### ❓ IF "I am learning a totally new concept..."
*   **THEN**: Use **Metaphor & Dual Coding**.
*   **Protocol**: Draw it. Compare it to something you know.
*   **Doc**: [`skills_memory_encoding_deep_dive.md`](file:///E:/wk%201-Learning%20science/skills_memory_encoding_deep_dive.md)

#### ❓ IF "I understand it, but I keep forgetting it..."
*   **THEN**: Use **Spaced Repetition + Interference Injection**.
*   **Protocol**: Wait until you generally almost forget it, then test yourself. Mix it with similar topics (Interleaving) to create "Desirable Difficulty."
*   **Doc**: [`skills_memory_retrieval_deep_dive.md`](file:///E:/wk%201-Learning%20science/skills_memory_retrieval_deep_dive.md)

#### ❓ IF "I am stuck/blocked on a hard problem..."
*   **THEN**: Use **Incubation / Representational Change**.
*   **Protocol**: Walk away. Shower. Sleep on it. Force a "10% Move" (make the problem worse).
*   **Doc**: [`problem_solving_science.md`](file:///E:/wk%201-Learning%20science/problem_solving_science.md)

#### ❓ IF "I am learning a motor skill / language..."
*   **THEN**: Use **Immersion & Shadowing**.
*   **Protocol**: High volume input. Mimicry (Mirror Neuron activation).
*   **Doc**: [`language_acquisition_science.md`](file:///E:/wk%201-Learning%20science/language_acquisition_science.md) or [`embodied_cognition_learning.md`](file:///E:/wk%201-Learning%20science/embodied_cognition_learning.md)
