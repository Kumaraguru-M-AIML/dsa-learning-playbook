# 🧠 LEARNING SCIENCE (From Notebooks)

## Learn How to Learn — Top 5 Principles

### 1. Retrieval Practice (Active Recall)
*   **What**: Force your brain to retrieve information instead of re-reading.
*   **How**: Close the book → Write everything you remember → Use flashcards.
*   **Why**: The struggle to recall signals importance → Strengthens neural pathways.

### 2. Spaced Repetition
*   **What**: Spread study over time, not cramming.
*   **How**: Review at increasing intervals (1d → 3d → 1w → 1m). Use Anki.
*   **Why**: Flattens the forgetting curve → Short-term → Long-term memory.

### 3. Deep Encoding (Elaboration)
*   **What**: Connect new information to existing knowledge.
*   **How**: Feynman Technique — Explain like teaching a child. Ask "Why?" and "How?"
*   **Why**: Builds schema (web of knowledge) → Information becomes "stickier."

### 4. Interleaving
*   **What**: Mix different topics within one session.
*   **How**: Don't do 2 hours of one topic. Mix related concepts.
*   **Why**: Teaches WHEN to apply strategies, not just HOW.

### 5. Metacognition (Self-Regulation)
*   **What**: "Thinking about your thinking."
*   **How**: Before: Set goals. During: "Am I understanding or just recognizing?" After: Reflect.
*   **Why**: Prevents "illusion of competence."

## Justin Sung — Expertise Accelerator

### Core Principle: Encoding > Retrieval
*   Most students focus on retrieval (spaced repetition) — this is "refilling a leaky bucket."
*   **The Fix**: Improve ENCODING (first-time learning) → Naturally retain more.

### Higher-Order Learning (Schemas)
*   Move from isolated facts (Lower Order) to connected networks (Higher Order).
*   Information becomes "stickier" when interconnected.

### Priming & Scaffolding
*   **Never jump into details first.**
*   Scan material → Build big-picture structure → Details have "hooks" to latch onto.

### Cognitive Load Management
*   **Desirable Difficulty**: If it feels too easy, you're not learning. If too hard, you're overloaded.
*   Stay in the optimal struggle zone.

### Self-Regulated Learning
*   Be the "mechanic of your own brain."
*   Adjust techniques in real-time based on what the material requires.

## White-Box Learning — The 3 Phases

### Black Box (Unconscious Incompetence)
*   You don't know WHY you succeed or fail.
*   Relies on blind trust, rote memorization, intuition.
*   Leads to inconsistent performance and burnout.

### Grey Box (Conscious Exploration)
*   You start identifying SOME variables.
*   You form hypotheses and test them.
*   Performance becomes more consistent.

### White Box (Full Transparency)
*   You understand ALL causal relationships.
*   You can predict outcomes before acting.
*   You execute known protocols for guaranteed results.

### The Transformation Protocol
1. **"Why?" Interrogation**: When you fail, ask "Why?" 3 times to find root cause.
2. **Hypothesis Loop**: Create predictions before practicing, then verify.
3. **Map the System**: Visualize Inputs, Environment, and Feedback channels.

## Calisthenics — The Big 4

### Core Movement Patterns
| Pattern | Targets | Key Exercise |
|---------|---------|--------------|
| **Push** | Chest, Shoulders, Triceps | Push-ups, Dips |
| **Pull** | Back, Biceps | Pull-ups, Rows |
| **Squat** | Quads, Glutes, Hamstrings | Squats, Lunges |
| **Core** | Abs, Lower Back | Planks, Leg Raises |

### Key Principles
*   **Grease the Groove (GtG)**: Practice 4-8x/day at 40-60% max reps. Builds neural pathways.
*   **Form > Intensity**: Sloppy reps = Poor motor habits.
*   **Progressive Overload via Leverage**: Shift body position to increase difficulty.
*   **10-Second Rule**: Don't progress until you hold current variation for 10 sec.

### Beginner Workout Structure
1. **Warm-Up** (5-10 min): Mobility + light cardio
2. **Skill Work** (10-15 min): Practice technical moves while fresh
3. **Strength Work**: Push (3-4 sets) → Pull (3-4 sets) → Legs (3 sets) → Core (3 sets)
4. **Cool Down**: Static stretching

## Parkour — Core Movements

### Fundamental Movements
| Move | Description |
|------|-------------|
| **Precision Jump** | Jump from one point to another with exact landing |
| **Safety Roll** | Diagonal roll to absorb impact |
| **Vault** | Pass over obstacle using hands (Speed, Kong, Lazy) |
| **Wall Run** | 2-3 steps on vertical wall to gain height |
| **Cat Hang** | Hang on wall with hands gripping top, feet on surface |
| **Climb-Up** | Transition from hanging to standing on wall |

### Safety Principles
*   **"Quiet" Landing**: Loud = Joints absorbing force. Soft = Muscles absorbing.
*   **Ukemi (Falling)**: Learn to fall safely before learning to jump.
*   **100-Rep Rule**: 100 reps on ground before adding 1-foot height.
*   **Environmental Assessment**: Check surfaces, stability, hazards FIRST.

## Flow State — Movement Mastery

### The Challenge-Skill Balance
*   Flow occurs when HIGH CHALLENGE meets HIGH SKILL.
*   Too easy = Boredom. Too hard = Anxiety.
*   **Adjust constantly**: Increase challenge as skill grows.

### Entering Flow
1. **Clear Goal**: Know exactly what you're trying to do.
2. **Match Difficulty**: Ensure challenge = current skill level.
3. **Relax the Body**: Tension blocks flow. Move from the center.
4. **Don't Stop for Mistakes**: Adapt and keep moving.

### Key Insights
*   **Focus on Transitions**: Mastery is revealed in the transitions between moves.
*   **"Happy Accidents"**: Treat errors as stepping stones, not setbacks.
*   **Effortless Attention**: High performance correlates with muscle relaxation.
