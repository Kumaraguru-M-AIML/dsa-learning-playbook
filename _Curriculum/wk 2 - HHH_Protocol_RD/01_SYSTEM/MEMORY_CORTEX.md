## 🔄 RAG RETRIEVAL PIPELINE (How I "Remember")

When you ask a question:
1.  **Query Analysis**: I determine if the query is **Fact-based** (Semantic) or **Historical** (Episodic).
2.  **Vault Search**: I pull the `MASTER__` protocol files.
3.  **Event Check**: I check if your previous data contradicts or supports the protocol.
4.  **Visual Recall**: I check if there is an image (e.g., an architecture diagram or a body-fat reference) associated with the query.
5.  **Response Synthesis**: I give you the answer integrated with your unique history.
