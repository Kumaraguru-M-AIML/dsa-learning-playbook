# PHASE 1 — STRUCTURAL RECOVERY (0–6 MONTHS)

## Priority Stats (Ranked)

```
1. Regeneration system (sleep, recovery habits)
2. Energy system (aerobic base — walking, Zone 2)
3. Movement system (mobility — CARs, squat holds)
4. Structural efficiency (fat reduction — mild deficit)
5. Basic neural activation (learning compound patterns)

Strength is SECONDARY here.
```

## Biological Objectives

```
→ Restore metabolic health
→ Reduce fat mass
→ Strengthen joints and tendons (tolerance phase)
→ Establish nervous system activation patterns
→ Improve cardiovascular function
```

## Measurable Targets by Month 6

```
✔ Lose 10–20% of excess body fat
✔ Resting heart rate reduced
✔ Perform:
    → 10 clean push-ups
    → 20 bodyweight squats
    → 30-second plank
✔ Walk 8–12k steps daily comfortably
✔ Deep squat without pain
✔ No chronic joint irritation
```

## Stage 0 Sub-Phase (0–3 Months: Damage Control)

```
Daily structure:
  → 8–12k steps per day
  → 2–3 full body strength sessions (basic compounds)
  → 2 aerobic sessions (Zone 2)
  → Daily mobility (10 min)

Expected outcome after 3 months:
  → 5–10 kg fat loss (depending on starting weight)
  → Noticeable strength increase
  → Better posture
  → Better energy
  → Lower resting heart rate
  → You become TRAINABLE
```

## Weekly Structure

```
Day 1 — Full body strength A (goblet squat, RDL, push-up, row)
Day 2 — Zone 2 cardio + mobility
Day 3 — Full body strength B (lunges, deadlift, press, pulldown)
Day 4 — Rest or light walking
Day 5 — Strength + coordination (step-ups, carries, Pallof)
Day 6 — Long easy cardio (45–60 min walk/cycle)
Day 7 — Full rest
```

## Training Parameters

```
All strength exercises:
  → 3–5 sets × 5–8 reps
  → Stop before failure (2 reps in reserve)
  → Rest 2–3 minutes
  → Perfect form always
  → Slow controlled descent
```

## Nutrition

```
→ Small calorie deficit (~300–400 kcal)
→ Protein each meal
→ Water high
→ Reduce ultra-processed foods
→ Sleep 7.5–9 hours
→ Consistency beats perfection
```

## Level 1 Benchmarks (You KNOW you're in Level 1 when...)

```
✔ Train 3–5 days weekly without breaks
✔ Log workouts or track progress
✔ Soreness decreases over time
✔ Movement feels smoother each month
✔ Resting heart rate slowly drops
✔ Rarely get injured
✔ Can predict recovery time
✔ Progress becomes PREDICTABLE, not random
```

## Biggest Risks at Phase 1

```
1. Doing too much too fast
2. Switching programs constantly
3. Chasing aesthetics instead of function
4. Ignoring recovery
5. Thinking intensity = progress

Phase 1 is about STABILITY, not intensity.
```

## Goal of Phase 1

```
Not strength. Not speed. Not dominance.

The goal is:
  "Become structurally reliable and consistently trainable."

Compounding CANNOT happen on an unstable base.
```

# PHASE 2 — STRUCTURAL STRENGTHENING (6–18 MONTHS)

## Priority Stats

```
1. Relative strength (force system, neural focus)
2. Tendon strength (slow eccentrics, isometrics)
3. Structural alignment (posture, symmetry)
4. Neural drive efficiency (compound lifts, low rep)
5. Continued fat optimization
```

## Biological Objectives

```
→ Increase neural recruitment capacity
→ Strengthen tendon structure
→ Continue body fat reduction
→ Build movement competence under load
→ Develop consistent training tolerance
```

## Measurable Targets by Month 18

```
✔ Relative strength clearly improved
✔ Body fat noticeably reduced
✔ 5–8 controlled pull-ups (or progressing toward)
✔ Stable single-leg balance 30+ seconds
✔ No chronic joint issues
✔ Can train 4–5 days weekly without burnout
✔ Recovering within 48 hours
```

## Training Evolution

```
→ Weights become challenging but controlled
→ Movement patterns feel natural
→ Progressive overload is consistent
→ Neural adaptation becomes primary driver
→ Begin introducing explosive primers (light jumps)
```

# PHASE 3 — NEURAL & POWER DEVELOPMENT (18–36 MONTHS)

## Priority Stats

```
1. Neural power conversion (strength → explosive output)
2. Speed introduction (short sprints, plyometrics)
3. High coordination work (rings, single-leg, rotational)
4. Structural symmetry (identify and correct imbalances)
5. Advanced recovery optimization
```

## Biological Objectives

```
→ Convert accumulated strength into SPEED
→ Develop explosive RFD (Rate of Force Development)
→ Refine coordination across all systems
→ Optimize body composition for power-to-weight
→ Build advanced recovery systems
```

## Measurable Targets by Month 36

```
✔ Relative strength HIGH
✔ Body fat LOW (near optimal)
✔ Tendons STRONG
✔ Reaction speed IMPROVED
✔ Energy system STRONG
✔ Sprint acceleration noticeably improved
✔ Explosive power (vertical jump, broad jump) improving
```

## What Changes in Training

```
→ Strength sessions: neural-focused (1–5 reps, heavy, long rest)
→ Speed sessions: short sprints (10–30m), full recovery
→ Plyometrics: low volume, high quality
→ Ring work and advanced bodyweight skills
→ Block periodization begins (rotating emphasis)

This is where "APEX TRAJECTORY" becomes visible.
```
