## 🏗️ THE SCAFFOLDED PATHWAY (4 STEPS)

### STEP 1: THE STRUCTURAL RESET (Weeks 1-4)
*Goal: Fix the frame so it doesn't break under load.*
- **Deep Dive Source:** *Convict Conditioning Lvl 1*, *6FT6 Method*.
- **The Protocol:**
    - **Dead Hangs:** 3 mins total/day (Decompresses spine).
    - **Wall Angels:** 3x10 (Fixes rounded shoulders).
    - **Glute Bridges:** 3x15 (Fixes ATP/Weak glutes).
    - **Micro-Skill:** Nasal Breathing only (During all exercise).

### STEP 2: THE CALISTHENIC FOUNDATION (Weeks 5-12)
*Goal: Mastery of the Bodyweight Vessel.*
- **Deep Dive Source:** *Convict Conditioning*, *Building the Gymnastic Body*.
- **The Protocol:**
    - **Pushup Progression:** From Wall -> Incline -> Flat.
    - **Pullup Progression:** From Vertical Pulls -> Horizontal Rows -> Negative Pullups.
    - **Squat Progression:** From Assisted -> Full Bodyweight.
    - **Frequency:** 3x per week. Every rep must be slow (3s down, 1s hold).

### STEP 3: THE HYPERTROPHY PUMP (Months 3-6)
*Goal: Adding functional armor (Muscle).*
- **Deep Dive Source:** *Jeff Nippard High Frequency*, *SuperFunctional Training*.
- **The Protocol:**
    - **Compound Lifts:** Low-weight Barbell Squats, Barbell Rows, Overhead Press.
    - **Volume:** 10-15 sets per muscle group per week.
    - **Fuel:** 1.2g Protein per lb. Egg-based cholesterol loading (for hormones).

### STEP 4: THE METABOLIC ENGINE (Months 6-9)
*Goal: High-torque output and endurance.*
- **Deep Dive Source:** *Sprinting Protocols*, *Hybrid Athlete*.
- **The Protocol:**
    - **Hill Sprints:** 6x30m (Full recovery). Spikes HGH.
    - **Zone 2 Cardio:** 45 mins steady state run (Low heart rate).
    - **Combat Readiness:** Shadow boxing (3x3 min rounds).
