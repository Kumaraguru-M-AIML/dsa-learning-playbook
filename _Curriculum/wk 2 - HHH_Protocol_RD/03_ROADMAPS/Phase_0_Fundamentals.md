# 🏁 PHASE 0: THE FUNDAMENTALS

# 1. SYSTEMIC FUNDAMENTALS
*Goal: 100% Data Integrity.*

### 1.1 The Daily Log
- **Requirement:** Every night before sleep, report progress to Aria.
- **The Basic:** If it isn't logged, it didn't happen. No log = 0 XP for the day.
- **Protocol:** Open `JOURNAL.md`, use the template, sync with Aria.

### 1.2 XP & Rank Logic
- **Requirement:** Understand the ROI (Return on Investment) of your actions.
- **The Basic:** Every action has an XP value (found in `REFERENCE.md`). 
- **Protocol:** Track your streaks. Combos are the fastest way to level up.

# 2. COGNITIVE FUNDAMENTALS
*Goal: Mental Control.*

### 2.1 The Focus Block (The 2-Minute Rule)
- **Concept:** (Clear) If a task takes < 2 min, do it now. The first 120s of any block are the "Momentum Gate."
- **Protocol:** Set a timer for 2 min. Just open the book/file. Once the clock hits 0, you are legally allowed to stop, but you usually won't.

### 2.2 Alpha-State Encoding
- **Concept:** (Bremer) High brainwave frequency (Beta) blocks deep memory formation. Mastering the [Alpha Gate](../05_USM_Hub/Knowledge_Base/Module_01_Concentration.md) is mandatory.
- **Protocol:** 5-min Buddhist Breathing + 20→1 Countdown Drill before any session.
- **Resource:** [USM Wiki Hub](../05_USM_Hub/USM_WIKI_INDEX.md)

### 2.3 Retrieval over Recognition (Feynman Feedback)
- **Concept:** (USM Module 04) Re-reading is an illusion. Testing is growth.
- **Protocol:** Close source. Write 3 key facts from memory. Teach the concept back to Aria to verify synthesis.

# 3. PHYSICAL FUNDAMENTALS
*Goal: Structural Integrity.*

### 3.1 The "Big 6" Baseline
- **Focus:** (Wade) Mastering Step 1 of all movements.
- **Movement:** 
  - Wall Pushups (Chest)
  - Shoulderstand Squats (Legs/Balance)
  - Vertical Pulls (Back)
  - Knee Tucks (Core)
  - Short Bridges (Spine)
  - Wall Headstands (Shoulders)
- **Requirement:** Focus exclusively on the **2-1-2 Tempo**. Build the tendons.

### 3.2 Mobility Basics
- **Focus:** Removing structural "kinks" (Fascia).
- **Protocol:** Morning "Cat-Cow" and "Deep Squat Hold" (30s) to reset lower-body fascia.

*Goal: Endocrine Baseline.*
